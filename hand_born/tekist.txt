テスト1
Python から本番JSONを更新できることの確認

目的
Python の SetAppPref(update_pref) で本番JSONを更新できること

実施方法
本番JSONを開いて、初期値を確認する
特に次の2項目を見る
ModelName
WorkProcedureName
Pythonコードを起動する
メニューで 1 を入力する
1: Python から ModelName, WorkProcedureName を更新
実行後に、本番JSONを直接開いて内容を確認する
期待結果
ModelName の Value が PYTHON_TEST_MODEL
WorkProcedureName の Value が PYTHON_TEST_WORK

テスト3
名称

Pythonロック中にC#がアクセスできないことの確認

目的

以下を確認する。

LockAppPref / UnlockAppPref
LockApplicationPreference / UnlockApplicationPreference
更新中や読み込み中に他方から変更できないこと
実施方法
Python側を起動する
python app_pref_python_prod.py
メニューで 2 を入力する
2: Python でロックして10秒保持
その10秒の間に、C#側を起動する
dotnet run
C#メニューで 2 を入力する
2: ロックできるか試すだけ
期待結果

C#側で

C#: Lock result = False
確認できること
Pythonロック中はC#がファイルに入れない
排他制御が成立している

テスト7
名称

C#更新をPythonが検知して読めることの確認

実施方法

dotnet run
C#メニューで 3 を入力する
ModelName, WorkProcedureName を更新
期待結果

Python: Wait and Read result
ModelName = CSHARP_TEST_MODEL
WorkProcedureName = CSHARP_TEST_WORK
確認できること
C#の更新をPythonが検知できる
Pythonの GetAppPref() が仕様どおりに動いている
